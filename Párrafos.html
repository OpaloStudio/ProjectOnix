Párrafos y saltos de linea en HTML


Párrafos:

Para declarar un párrafo en HTML se utiliza el tag < p >


Para alinearlo se utiliza (aling= "left") después de p para alinearlo a la izquierda

Para alinearlo se utiliza (aling= "center") después de p para alinearlo en medio

Para alinearlo se utiliza (aling= "right") después de p para alinearlo a a derecha


Saltos de línea:

Para hacer un salto de línea se utiliz el tag < br > en donde se deseé el salto y no es necesario cerrar el tag:

"Este escrito

no contiene salto de línea en ninguna sección."


"Pero este escrito


si tiene un salto."


Tag < pre >:

Todo lo que esté escrito dentro del tag < pre > sera mostrado exactamente igual con sus respectivos saltos de linea y espacios.


            Por lo que puedo escribir
                        algo así
                                         y se mostrará todo en la misma posición
  siempre que este dentro del tag < pre >.

Código:

                < p > Todo lo que esté escrito dentro del tag < pre > sera mostrado exactamente igual con sus respectivos 
                    saltos de linea y espacios. < /p > < br >
                    < pre >
                        Por lo que puedo escribir
                                    algo así
                                                     y se mostrará todo en la misma posición
              siempre que este dentro del tag < pre >.                                       
                    < /pre > < br >
