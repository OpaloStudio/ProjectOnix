Listados en HTML


Para hacer listados en HTML se pueden usar los tags < ul > para listados sin orden y cada elemento de la lista lleva el tag < li > :

  • Papa
  • Tomate
  • Pollo

Y también se puede utilizar el tag < ol > para listas ordenadas y cada elemento de la lista lleva el tag < li > :
  1. Papa
  2. Tomate
  3. Pollo
